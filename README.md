# Portfolio Website

This is a personal portfolio website built with React. It showcases my projects, skills, and experience in a visually appealing and interactive way.

## Features

-   **Interactive UI:** The website is designed to be engaging, with animations and interactive elements.
-   **Component-Based Architecture:** Built with React, the codebase is organized into reusable components.
-   **Project Showcase:** A dedicated section to display my projects with descriptions and links.
-   **Skills and Experience:** Detailed sections for my technical skills, tools, and professional experience.
-   **Contact Form:** An integrated contact form for reaching out to me.
-   **Resume Viewer:** An interactive PDF viewer for my resume.

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes.

### Prerequisites

You need to have Node.js and npm installed on your machine. You can download them from [here](https://nodejs.org/).

### Installation

1.  **Clone the repo**
    ```sh
    git clone https://github.com/your-username/your-repo-name.git
    ```
2.  **Navigate to the project directory**
    ```sh
    cd your-repo-name
    ```
3.  **Install NPM packages**
    ```sh
    npm install
    ```

## Usage

### Running the development server

To run the app in development mode, use the following command:

```sh
npm start
```

This will open the app at [http://localhost:3000](http://localhost:3000) in your browser. The page will reload if you make edits.

### Building for production

To build the app for production, use the following command:

```sh
npm run build
```

This will build the app for production to the `build` folder. It correctly bundles React in production mode and optimizes the build for the best performance.
